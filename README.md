# 一念成歌

AI 歌词、作曲、乐谱、钢琴 Demo 生成网站。

## 功能

- 用户输入灵感/主题/风格/情绪，AI 自动生成高质量中文歌词
- 根据歌词自动生成作曲方案（BPM、调式、和弦、旋律方向）
- 生成简谱 + 歌词对齐的词曲乐谱
- 生成钢琴 Demo 提示词（后续接入音频生成）

## 技术栈

- 前端：HTML / CSS / JavaScript（纯静态，零框架）
- AI 后端：智谱大模型 API（GLM-4）
- 部署：静态托管（Vercel / Netlify / GitHub Pages）

## 本地运行

1. 克隆仓库
2. 在根目录创建  文件，填入智谱 API Key
3. 用浏览器打开  即可预览原型
